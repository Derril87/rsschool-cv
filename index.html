KATA from LeetCode - Two sum:

In this task, we are given an array of integers and a target integer. The goal is to find two numbers in the array that add up to the target and return their indices. 


               function(nums, target) {
                 const map = new Map();
                 for (let i = 0; i < nums.length; i++) {
                   let diff = target - nums[i];
                   if (map.has(diff)) {
                     return [i, map.get(diff)];
                   }
                   map.set(nums[i], i);
                 }
                 return [-1, -1];
               }
